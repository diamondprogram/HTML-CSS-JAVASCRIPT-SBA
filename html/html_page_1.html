what is html and what is an html file?

Html stands for Hyper Text Markup Langauge. Html is a mark-up langauge and not a programing langauge. What dose it mean to be a "Markup Langauge", which is just a fancy way of saying a text file with some identifier tags. Yes, the webpages you see on the internet are really just a type of text file.
